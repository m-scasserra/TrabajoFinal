\doxysubsection{Detailed Description}
This class provides an interface for GPIO and ADC operations. 

The {\ttfamily \doxylink{class_i_o}{IO}} class is a singleton class that allows access to GPIO and ADC functionalities. It provides methods for initializing GPIOs, configuring GPIOs, setting and getting GPIO levels, configuring the ADC, and reading raw and converted values from the ADC.

The class follows the Singleton design pattern to ensure that only one instance of {\ttfamily \doxylink{class_i_o}{IO}} can exist. To access the instance, use the static method {\ttfamily get\+Instance()}.

The class also prevents copying and assignment of instances by deleting the copy constructor and the assignment operator.
