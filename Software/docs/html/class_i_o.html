Detailed Description

This class provides an interface for GPIO and ADC operations.

The IO class is a singleton class that allows access to GPIO and ADC functionalities. It provides methods for initializing GPIOs, configuring GPIOs, setting and getting GPIO levels, configuring the ADC, and reading raw and converted values from the ADC.

The class follows the Singleton design pattern to ensure that only one instance of IO can exist. To access the instance, use the static method getInstance().

The class also prevents copying and assignment of instances by deleting the copy constructor and the assignment operator.
